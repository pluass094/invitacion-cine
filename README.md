# Invitacion al cine

Invitacion web para ver **Spider-Man: Brand New Day**.

## Archivos principales

- `index.html`
- `style.css`
- `script.js`
- `assets/SM.jpg`
- `assets/gone-gone-gone.mp3`

## Publicacion en GitHub Pages

Sube todos los archivos de esta carpeta a un repositorio publico de GitHub.
Luego activa GitHub Pages desde `Settings > Pages`, usando la rama `main` y la carpeta `/root`.
